# Pizza Sales Analysis Dashboard
This dashboard provides a comprehensive view of sales performance, customer demand, and product insights, with interactive filters that allow users to dynamically explore trends across time periods, pizza categories, and sizes, making it easy to analyze specific segments of the business.
## Key Features

- Interactive filters for month, pizza category, and pizza size
- Overview of key business metrics such as revenue, orders, and average order value
- Trend analysis with daily and monthly views
- Product mix analysis by category and size
- Ingredient-level insights using treemap and heatmap visualizations
- Demand analysis by time of day and product category
- Dynamic Conclusion & Key Insights section that updates based on selected filters
## Dashboard Sections

### Overview
Displays key performance indicators including total revenue, total orders, pizzas sold, average order value, and the top-performing pizza.

### Trends
Analyzes revenue and order trends over time with the ability to switch between daily and monthly views.

### Product Mix
Explores revenue contribution by pizza category and size, along with ingredient usage insights to understand menu composition.

### Demand Pattern
Focuses on customer behavior, highlighting top ingredients, best-selling pizzas, demand across different times of day, and category-level demand patterns.

### Conclusion & Key Insights
Summarizes the most important insights dynamically based on selected filters, translating data into actionable business takeaways.
